# Robot
Self-driving robot that will avoid obstacles

Components used:
1. Arduino Nano
2. 9V battery cage
3. HC-SR04 Ultra-sonic range finder
4. 128x64 Blue I2C OLED Display 
5. 2 x Continous Servo motors
6. SunFounder Digital Compass HMC5883L
7. Breadboard
